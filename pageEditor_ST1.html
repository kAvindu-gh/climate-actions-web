My Role

I was responsible for designing and implementing the Splash Screen, Volunteer Page, and a Content Page about Deforestation.

Pages I Created

  • Volunteer Page

What I Did

The splash screen features a slideshow, a spinning loader, and a redirect timer. The volunteer page includes scrollable cards and a feedback form. The deforestation page explains causes, effects, and solutions, with internal navigation and scroll-to-top.

Challenges

Getting the slideshow and scroll-to-top feature to work smoothly with styles was a bit tricky at first, but I fixed it using JavaScript and styled CSS animations.

Accessibility

All images have alt text, navigation is clear, and colors have strong contrast for readability.
